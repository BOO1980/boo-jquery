JQuery Selectors

  • The JQuery element selects elements based on the element name
  • You can select elements on a page
  • The ID selector used the id attribute of an HMTL tag to find the specific element
  • An ID should be unique within a page, so you should use the #ID selector to find a single unique elemtent
  • To find an element with a specific ID, write a hash character, followed by the ID of the HTML element
