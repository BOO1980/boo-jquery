JQuery Overview

  • JQuery is a JavaScript library
  • JQuery greatly simplifies JavaScript programming
  • JQuery is a lightweight JavaScript library
  • JQuery wraps lines of JavaScript code into methods that you can call with a single line of code
  • JQuery simplifies a lot of the complicated things from JavaScript, like AJAX call and DOM manipulation
  • The JQuery library contains the followng features
    • HTML/DOM manipulation
    • CSS manipulation
    • HTML event methods
    • Effects and animations
    • AJAX
    • Utilities
  • JQuery had pluggins for almost any task out there
  • The JQuery library is a single JavaScript file which you can reference it with HTML script tag
  • To include JQuery in your webpage you can download it from CDN or reference it from google api in the script tag ("https://ajax.googleapis.com/ajax/libs/jquery/3.6.4/jquery.min.js)
  • JQuery syntax is tailer made for selecting HTML elements and performing some action on the elements(s)
  • Basic syntax is $(selecto).action()
    • A $ sign to define/access JQuery
    • A (selector) to "query (or find)" HTML elements
    • A JQuery action() to be performed on the element(s)
